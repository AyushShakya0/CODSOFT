Food Delivery App

This project is a comprehensive food delivery application developed using Laravel for the backend and React for the frontend. The app allows users to browse through a variety of restaurants and dishes, place orders, and track delivery status in real-time.

Features include:

  • User authentication and authorization
  • Responsive user interface for browsing menus and placing orders
  • Real-time order tracking and notifications
  • Admin panel for managing restaurants, menus, and orders
  • Integration with payment gateways for secure transactions

This project showcases my skills in full-stack development, focusing on creating a seamless user experience and efficient backend operations.

Line Following Robot

This project involved building a line-following robot using Arduino and IR sensors. The robot is designed to follow a black line on a white surface, making it suitable for various automation and robotics applications.

Features include:

  • IR sensors for line detection and tracking
  • Arduino-based control system for precise navigation
  • Algorithm for adjusting the robot’s movement based on sensor input
  • Integration of motor controls for smooth operation

This project demonstrates my hands-on experience with robotics and electronics. It showcases my ability to implement algorithms and control systems, as well as my proficiency in working with Arduino and sensor technology.
